Instructions:
This module changes any taxonomy term link to be used as a link to a Apache Solr search filter.
To Change the taxonomy link:


2. Click the "manage display" tab for the content type you desire to change the term render.
   Example: articles admin/structure/types/manage/article/display
3. Locate the taxonomy term and Select Format, Apachesolr Link
4. Optional: Adjust the path and target with the cogwheel to the right.
